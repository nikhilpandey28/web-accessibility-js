# Web Accessibility in JavaScript

This project demonstrates basic web accessibility concepts using HTML, CSS, and JavaScript.

## Tasks Completed

### Task 1
- Created a native `<button>` and a plain `<div>`.
- Logged their `tabIndex` values.
- Verified:
  - Button → `0`
  - Div → `-1`

### Task 2
- Added:
  - `role="button"`
  - `tabindex="0"`
- Confirmed the attributes through JavaScript.

### Task 3
- Built a toggle button using:
  - `aria-expanded`
  - `aria-controls`

### Task 4
- Clicking the button:
  - Updates `aria-expanded`
  - Shows/hides the panel

### Task 5
- Added an accessible name to an icon-only button using `aria-label`.

### Task 6
- Associated a `<label>` with an `<input>` using:
  - `for`
  - `id`
- Verified the association in JavaScript.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes

- Native HTML accessibility
- Keyboard focus (`tabIndex`)
- ARIA roles
- ARIA states
- Accessible names
